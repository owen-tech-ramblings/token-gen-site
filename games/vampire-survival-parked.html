Current State

The current build is a standalone HTML canvas game with a large city map, camera, three warding crosses, Captain Voss, enemy roles, feed/dash/mist/Bat Swarm abilities, minimap, compass, high scores, dawn timer, blood rose pickups, combo scoring, particles, screen shake, and procedural audio cues.

Iteration 32 adds modular editable source, a deterministic self-contained build, profile schema v2, one-time v31 migration, corrupt-save recovery, revision checks, idempotent progression events, and automated regression contracts.

Development Workflow

  1. Edit files under games/vampire-survival-src/.
  2. Generate the live file and exact archive with node tools/build-vampire-survival.mjs --archive.
  3. Run profile, site contract, deterministic build, browser, and responsive checks.
  4. Publish each accepted iteration and verify it live before beginning the next.
